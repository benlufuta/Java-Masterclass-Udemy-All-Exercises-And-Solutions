# Udemy Java Masterclass All Exercises and Solutions

This repository contains my solutions for coding exercises from the Udemy course "Java Masterclass 2025: 130+ Hours of Expert Lessons" by Tim Buchalka.The course offers a thorough overview of Java, from basic concepts to advanced topics. The contents of this course were last updated on 12/2025.


## How to Use

- Each exercise directory contains the Java source code files for the solution and excercise description.txt file.
- Review the code and description file to understand the requirement and implementation.
- Use any Java development environment to run and test the solutions.
